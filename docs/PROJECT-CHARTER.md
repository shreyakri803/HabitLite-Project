## 1. Executive Summary
HabitLite is a minimal, fast, and lightweight habit-tracking solution designed to help users build and maintain daily routines.  
This project demonstrates full-cycle project management execution including planning, development coordination, documentation, testing, and release.

## 2. Project Background
Many habit-tracking apps overwhelm users with too many features.  
HabitLite focuses on simplicity, speed, and ease of use.  
This project also serves as a professional PM case study showing structured work and documentation.

## 3. Project Objectives
- Deliver a clean MVP with the ability to add, edit, delete, and complete daily habits.
- Store all habit activity using browser LocalStorage.
- Provide a weekly summary view and CSV export.
- Complete the project in a 3-week structured plan (simulated timeline).

## 4. Key Deliverables
- Requirements documentation

- Gantt timeline

## 5. Success Criteria
- All MVP features functional and tested.
- < 20 open bugs at launch.
- At least 5 beta testers complete feedback.
- Documentation fully completed and stored in GitHub.

## 6. Scope Definition
### In Scope
- Daily habit UI  
- LocalStorage saving  
- Weekly summary  
- CSV export  
- Basic analytics (streak counter)

### Out of Scope
- Cloud sync  
- Login/authentication  
- Push notifications  
- Multi-user setup  
- Paid features
